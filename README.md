# Epoch

A Digital Clock written from ground up in C++ using ncurses

_Clone:_  
`git clone https://github.com/azimex/Epoch.git`  
_Make:_  
`cd Epoch; make;`  
_Run:_  
`make run;`

_Run with options_  
`./time 2 5 "+%M:%S"`  
arg1 = font  
arg2 = color (0-7)  
arg3 = date format
